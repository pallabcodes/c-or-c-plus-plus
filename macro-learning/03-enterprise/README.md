# Enterprise Macro Patterns

## Examples

### 01_linux_kernel_macros.c
Demonstrates Linux kernel macro patterns:
- container_of macro for type-safe pointer arithmetic
- ARRAY_SIZE for compile-time array size
- BUILD_BUG_ON for compile-time assertions
- min/max macros with type safety
- likely/unlikely for branch prediction
- Alignment macros

### 02_system_library_patterns.c
Demonstrates system library macro patterns:
- Error handling macros (glibc-style)
- Assertion macros with custom messages
- Logging macros with levels
- Resource management macros
- Configuration macros
- Memory allocation macros

## Key Concepts
- Production-tested patterns from Linux kernel
- System library macro conventions
- Error handling and logging patterns
- Resource management patterns

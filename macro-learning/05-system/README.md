# System Programming Macros

## Examples

### 01_conditional_compilation.c
Demonstrates conditional compilation:
- Platform detection (Linux, macOS, Windows)
- Architecture detection (x86_64, ARM, etc.)
- Compiler detection (GCC, Clang, MSVC)
- C standard detection (C11, C99)
- Feature detection
- Debug build detection
- Build configuration macros

## Key Concepts
- #if, #ifdef, #ifndef directives
- Platform-specific code
- Feature detection
- Build configuration
- Conditional compilation patterns

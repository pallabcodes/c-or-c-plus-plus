# Advanced Macro Techniques

## Examples

### 01_x_macros.c
Demonstrates X-macro technique:
- Define data once, generate multiple code patterns
- Enum generation from X-macro
- String array generation
- Switch statement generation
- Error code system example

### 02_type_generic_macros.c
Demonstrates C11 _Generic keyword:
- Type-based dispatch in macros
- Type-safe generic operations
- Type-generic print macro
- Type-generic max macro
- Format specifier selection

## Key Concepts
- X-macros for code generation
- Single source of truth pattern
- C11 _Generic for type safety
- Compile-time type selection

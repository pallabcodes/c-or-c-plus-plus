# Macro Fundamentals

## Examples

### 01_object_like_macros.c
Demonstrates basic object-like macro definitions:
- Constants and configuration values
- Feature flags
- Compile-time calculations
- Type definitions via macros

### 02_function_like_macros.c
Demonstrates function-like macros with proper parenthesization:
- MIN/MAX macros
- ABS and SQUARE macros
- ARRAY_SIZE macro
- OFFSETOF macro
- Importance of parentheses for operator precedence

### 03_include_guards.c
Demonstrates include guard patterns:
- Traditional #ifndef pattern
- Modern #pragma once alternative
- Preventing multiple inclusion
- Preventing redefinition errors

## Key Concepts
- Object-like vs function-like macros
- Macro expansion rules
- Parenthesization importance
- Include guard patterns
